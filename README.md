# ssm-learn-go
这是我学习 Golang 的练手项目，至于为什么是学生成绩管理系统？这不是当时大一学 C 的时候必备项目嘛 hhhhh

# 迭代路线图
V 1.0 LinkedList
1. 维护链表实现 CRUD 且支持排序
2. 带有基本的输入检查
3. 使用交互式 Shell 方式操作程序
4. 基本数据结构：姓名、性别、生日 、年龄、可变科目的分数
5. 添加单元测试

V 2.0 MySQL
1. 使用 MySQL 作为数据库存储数据
2. 建表、清理数据、导入导出等脚本
3. 导入支持数据合并
4. 添加自动化测试，并附带测试报告

V 3.0 RESTful
1. RESTful API 实现
2. 不使用现成的 web 框架
3. 支持用户管理，创建更新删除
4. 支持上传下载学生信息（导入导出）
5. 更新自动化测试

V 4.0 RESTful client
1. 客户端 客户端 客户端
2. 可以登陆登出
3. 学生信息的 CRUD
4. mock 数据，便于没有服务端的时候测试
5. 更新自动化测试
